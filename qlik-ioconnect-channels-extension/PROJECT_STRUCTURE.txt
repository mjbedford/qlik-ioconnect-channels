## ğŸ“¦ What Gets Bundled

The webpack build bundles:
- Your extension code (src/main.js)
- @interopio/browser library (~150KB)
- AMD module wrapper (for Qlik compatibility)

Excluded from bundle (provided by Qlik):
- jQuery
- qlik (Qlik API)
- Angular
- Underscore

Final bundle size: ~200-300 KB (minified)

## ğŸ”Œ Integration Points

Extension publishes data in this format:
{
  "source": "Qlik Sense",
  "timestamp": "ISO-8601",
  "rowIndex": 0,
  "data": {
    "Field Name": {
      "text": "Display Value",
      "number": 12345
    }
  }
}

Other apps can subscribe:
io.channels.subscribe('QlikData', (data) => {
  console.log(data);
});
